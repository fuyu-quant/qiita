# Qiita


* 新しい記事の作成
```
npx qiita new 記事のファイルのベース名
```

* 記事の投稿
```
# 個別に記事の投稿を行う
npx qiita publish 記事のファイルのベース名
# 全ての記事を反映させる
npx qiita publish --all
```

* Qiitaとの同期
```
# Qiita上で更新を行い手元で変更をしていない記事を同期させる
npx qiita pull
```
